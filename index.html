Projects

Deepfake Video Detection

This is a machine learning-based system built using Python that detects and flags manipulated video content. Leveraging OpenCV, TensorFlow/Keras, and dlib, the system uses convolutional neural networks (CNNs) to identify visual anomalies in facial movements and inconsistencies commonly found in deepfake videos. It will developed and tested using DFDC, and training was performed on Google Collab. The application aims to enhance digital media authenticity and combat misinformation.

Student Result Management System

A web application developed using HTML, CSS, JavaScript, and PHP, integrated with a MySQL database. This system streamlines the process of managing and publishing academic results for students. It features an admin portal for entering and updating results, and a student portal to securely view grades using unique login credentials. Hosted on a local XAMPP server and designed with Bootstrap and Figma.

Personal Task Manager

A SPA built using React.js for efficient task tracking and management. Users can create, edit, delete, and filter daily tasks with real-time updates powered by React Hooks (useState, useEffect). The UI is designed using Bootstrap. it showcases skills in component-based development and modern front-end technologies.

Simple Weather App

React-based app that fetches weather data from an API and displays real-time conditions.
